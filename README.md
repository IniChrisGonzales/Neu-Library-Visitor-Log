# 📚 NEU Library Visitor Logger

**Live Application:** [https://neu-library-log-491cc.web.app/](https://neu-library-log-491cc.web.app/)

A modern, web-based application designed to streamline library check-ins, track visitor analytics, and manage student access in real-time. Built with React and Firebase, this system replaces traditional paper logs with a clean, SaaS-style digital interface. 

## ✨ Key Features

* **Seamless Student Check-in:** A fast, intuitive portal for students to log their library visits.
* **Real-Time Admin Dashboard:** Instantly view daily visitor counts, active users, and key library statistics through elegant, data-driven metric cards.
* **Comprehensive Visitor Logs:** A digital ledger of all library entries, allowing admins to track and monitor facility usage accurately.
* **Role-Based Access Control (RBAC):** Secure routing and distinct permissions tailored for Students, Admins, and Super Admins.
* **Advanced User Management:** Admins can manually register new student accounts, monitor user statuses, and quickly block or unblock access to maintain library security.
* **Modern SaaS UI:** A fully responsive, light-theme design built with Tailwind CSS, featuring subtle shadows, clean typography, and polished micro-interactions.

## 🛠️ Tech Stack

* **Frontend:** React, Tailwind CSS, Lucide React (Icons), React Router
* **Backend & Database:** Firebase (Authentication & Firestore)
* **Hosting:** Firebase Hosting
